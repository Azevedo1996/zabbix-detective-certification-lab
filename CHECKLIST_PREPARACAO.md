# Checklist de preparação

## Ambiente

- [ ] Docker Desktop aberto e com engine Linux iniciada.
- [ ] Projeto iniciado pelo `iniciar.bat` sem erro.
- [ ] Zabbix Detective acessível em `http://localhost:8989`.
- [ ] Zabbix Lab acessível em `http://localhost:4000`.
- [ ] Containers principais em execução: Django, Zabbix Web, Zabbix Server, banco e agent.

## Antes de estudar

- [ ] Escolhi uma certificação ou uma missão extra para o bloco de estudo.
- [ ] Abri o Guia / Plano de estudo da certificação correspondente.
- [ ] Li a documentação oficial do tópico antes de responder.
- [ ] Sei diferenciar coleta, processamento, visualização, alerta, segurança e automação.

## Durante o laboratório

- [ ] Li o cenário completo.
- [ ] Executei o passo a passo no Zabbix Lab quando aplicável.
- [ ] Validei o resultado esperado.
- [ ] Registrei evidência ou anotação do que foi feito.

## Antes de conferir o gabarito

- [ ] Respondi com meu próprio raciocínio técnico.
- [ ] Comparei minha resposta com a explicação.
- [ ] Revisei a documentação oficial se houve dúvida.
- [ ] Só marquei como concluído após validar o checklist do desafio.

## Revisão semanal

- [ ] Revisei erros ou itens incompletos.
- [ ] Refiz ao menos um laboratório por certificação estudada.
- [ ] Consultei o gabarito por certificação ou por missão extra.
- [ ] Atualizei minhas anotações para a próxima semana.
