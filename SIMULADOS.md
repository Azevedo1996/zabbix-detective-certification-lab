# Links para Simulados e Provas — Zabbix Detective

Este arquivo reúne links úteis para simulação, inscrição em exames e prática adicional.

> **Atenção:** os links oficiais da Zabbix devem ser a principal referência. Simulados de terceiros podem ajudar na prática, mas não são material oficial da Zabbix e podem conter erros, conteúdo desatualizado ou alegações não verificáveis. Use-os apenas como complemento.

## Simulados internos do projeto

O próprio jogo pode ser usado como simulado local:

- **Simulado ZCU:** faça a fase ZCU Core sem consultar o gabarito.
- **Simulado ZCS:** faça a fase ZCS Core + Data Collection + Alerts + Users.
- **Simulado ZCP:** faça ZCP Core + Automação API + Ansible + LLD.
- **Simulado ZCE:** faça ZCE Core + Segurança + Autenticação + AWS + Azure + Bancos.
- **Simulado completo:** faça todas as missões principais e extras.

### Modelo recomendado de simulado interno

```text
ZCU -> 20 questões em 30 minutos
ZCS -> 50 questões em 60 minutos
ZCP -> 75 questões em 90 minutos
ZCE -> 90 a 120 minutos de revisão prática por cenários
```

## Como usar simulados de terceiros com segurança

1. Priorize sempre a documentação oficial.
2. Desconfie de materiais que prometem “questões oficiais” copiadas.
3. Use simulados externos para medir raciocínio, não para decorar.
4. Sempre valide respostas na documentação oficial.
5. Se houver divergência entre simulado externo e documentação oficial, confie na documentação oficial.

## Checklist antes de marcar exame

- [ ] Acerto acima de 85% nos desafios da certificação alvo.
- [ ] Erros revisados com base nos links oficiais.
- [ ] Capacidade de explicar respostas sem consultar gabarito.
- [ ] Pelo menos um simulado cronometrado concluído.
- [ ] Laboratórios práticos feitos para os tópicos mais fracos.
