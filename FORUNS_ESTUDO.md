## Como pedir ajuda corretamente

Ao abrir uma dúvida em fórum ou comunidade, inclua:

```text
Versão do Zabbix:
Sistema operacional:
Tipo de instalação:
Banco de dados:
Proxy ou server direto:
Sintoma:
Mensagem de erro:
Trecho de log:
O que já foi testado:
Print/tela, se aplicável:
```

## Regras de ouro

- Nunca publique senhas, API tokens, PSK, private keys ou secrets.
- Remova IPs públicos e nomes internos sensíveis se a política da empresa exigir.
- Informe a versão exata do Zabbix.
- Explique o objetivo, não apenas o erro.
- Depois de resolver, documente a solução no seu caderno de estudos.
