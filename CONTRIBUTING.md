# Contribuindo

Obrigado por considerar contribuir com o Zabbix Detective: A Missão.

## Como contribuir

1. Abra uma issue descrevendo a melhoria ou correção.
2. Crie uma branch a partir de `main`.
3. Faça alterações pequenas e bem descritas.
4. Teste o projeto localmente.
5. Envie um pull request explicando o que foi alterado.

## Padrão para novos desafios

Novos desafios devem conter:

- cenário claro;
- resposta sem ambiguidade;
- pista ao errar;
- explicação em estilo certificação;
- link oficial ou fonte confiável;
- sem copiar questões oficiais de exames.

## Execução local

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py runserver
```
