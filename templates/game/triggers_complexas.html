Modelo mental para criar uma trigger boa

  1. Qual item mede o sintoma?
  2. Qual janela de tempo elimina ruído?
  3. Qual limite representa impacto real?
  4. Existe recuperação diferente do limite de disparo?
  5. Existe dependência que deve suprimir o problema?
  6. Quais tags vão ajudar filtro, ação e correlação?
  7. Como validar o problema no Latest data e em Problems?
